CareerPilot – Smart Resume Analyzer

This project analyzes any resume (PDF) and provides:

- ATS Score (out of 100)
- Target Role Prediction
- Experience Level Detection
- Smart Suggestions for improvement

Features
- Upload resume in PDF format
- Detects technical & non-technical skills
- Identifies experience level (Fresher / Experienced)
- Calculates ATS-style score
- Provides actionable feedback and suggestions

 How It Works
- User uploads a resume (PDF)
- Text is extracted using pdfplumber
- Keywords and sections are analyzed
- Score, role, and suggestions are generated
  
Tech Used
- Python
- Gradio
- pdfplumber

How to Use
- Upload your resume (PDF)
- Click analyze
- Get instant report
